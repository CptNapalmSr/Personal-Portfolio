FrontEnd Project – Personal Portfolio Website
1. Short Introduction About Me
Hello! My name is Mikael Hylen, a project leader and software engineer based in Malmö, Sweden. I have
extensive experience in web and mobile application development, working with technologies such as Java,
JavaScript, RESTful services, and modern frontend frameworks. Over the years, I have led and contributed
to projects across finance, transportation, telecommunications, utilities, and reporting systems. I enjoy
building clean, user-focused applications and continuously improving my technical and leadership skills.
2. Project Title
Personal Portfolio Website
3. Project Description
This project is a personal portfolio website designed to showcase my professional background, technical
skills, and selected projects. The website serves as a central place where recruiters, collaborators, and peers
can quickly learn about me, explore my work, and get in touch.
The portfolio focuses on: - Clear presentation of information - Modern, responsive design - Clean and
maintainable frontend code
